# C++中整数按进制转换

>将一个十进制转换成二进制、八进制、十进制等

## 将一个十进制数转成d进制数

```C
  int nums[10001];
  int N, d;                  // N为一个十进制数，d为要转换的进制
  int i = 0;
  while (N) {
    nums[i++] = N % d;      //取模后的结果为十进制的低位 存入数组
    N /= d;                 //取模后剩余的N = N / d;
  }
  //举例：
  73 转10进制后数组里为 [7, 3]
  23 转2进制后数组里为 [1, 0, 1, 1, 1]
  //知识点：将一个十进制按进制转换存入数组
```

## 将转换后的数倒序输出

```C
  //一个十进制数按进制转换到数组里后倒序输出结果
  73 转10进制后数组里为 [7, 3]， 倒序输出为37
  23 转2进制后数组里为 [1, 0, 1, 1, 1]， 倒序输出为11101
  //代码可以这样写
  int k = 0, len = nums.size();
  int sum = 0;
  for (int j = len - 1; j >= 0; j--) {
    sum += nums[j] * pow(d, k);              //从最后一位起加起来，d为要当前值的进制
    k++;                                     //如果不是倒序求结果，则从j=0开始求加和
  }
  //知识点：数组里各位数求和得到新值
```
